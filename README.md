# fbox

http://asciii.fixxel.org/a/45

### How to

Modify the ***fboxfile*** so it matches to your need.

#### Bring the box up :
```
./fbox up
```

#### Stop the box :

```
./fbox halt
```

#### Reload the box :

```
./fbox reload
```

#### SSH the box :

```
./fbox ssh
```

#### Destroy the box :

```
./fbox destroy
```
